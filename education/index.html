Mirobot is made to teach

Mirobot is unique in how it takes a cross-curricular approach to learning. By tying together different subject areas, childrens' learning is strengthened as one subject reinforces another.

Design & Technology

Learn about the mechanics that make Mirobot move and how everything fits together. Tie in to robotics in the curriculum and learn what makes it go.

Computing

Use the range of existing apps to teach children about the basic principles of computing. Start with a simple visual programming tool and work up to using real languages like Javascript.

Mathematics

Using turtle robotics to draw geometric shapes is a classic way of getting children engaged in thinking about physical mathematical problems.

Art

Use Mirobot to get your class creating computer-controlled artwork.

Mirobot is designed to be easily disassembled so that it can be used with many different classes. It's robust, but easy to repair if it does get broken. All parts are interchangeable and can be replaced.

An app for every occasion!

There are apps available covering most skill levels, from basic point-and-click apps for beginners to more involved drag-and-drop block-based apps. Here's a selection, but you can see them all at apps.mirobot.io. All apps are completely free, cross platform and run in your web browser which means there's no software that needs installing before you can use Mirobot.

Point and Click

Use this app to let young children control Mirobot and make it draw things. Great for building up spatial awareness skills.

Blockly

A good first step towards programming Mirobot. Use this simple block based programming language to build an understanding of things like loops and other simple control structures.

Snap!

Snap! is a Scratch clone that runs in your browser which lets children tackle more complex programming challenges like line following or collision detection.

Javascript

Once children have progressed beyond more visual programming languages, Javascript is a good next step which will keep them challenged.
