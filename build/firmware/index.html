--- layout: default title: Customise the firmware ---

Mirobot comes ready to roll, but if you want to change things here's how to customise the Arduino code that runs at the heart of every Mirobot.

Step one

Make sure you've got the Arduino IDE installed on your computer. If you haven't, download it and install it.

Step two

Install the libraries we need to be able to run the code. These can be downloaded from:

  • The Mirobot Arduino library
  • The HotStepper stepper motor library
  • The PWMServo servo library

Once you have downloaded them, they need installing. Go to the menu Sketch >> Import Library >> Add Library and when prompted select the zip file you just downloaded

You can tell that they are installed properly because when you go to Sketch >> Import Library again they will be listed

If you are having problems installing them, see here for more details

Step three

In your File >> Examples menu, you should have a menu item for Mirobot (if you haven't, then the library is not installed properly). Select "socket_example" and it should open up a short sketch in the Arduino IDE.

Step four

Plug Mirobot into your computer using the USB <-> Serial adapter. If you don't have one of these you can get one from: TODO

Select the port from the Tools >> Serial Port menu

Select "Arduino Duemilanove w/ ATmega328" from the Tools >> Board menu

Click on the upload icon or choose File >> Upload from the menu. You should see a success message saying that the firmware has been successfully compiled and uploaded to Mirobot

Now you can either use it as normal or start from here customising the firmware to do your bidding!
